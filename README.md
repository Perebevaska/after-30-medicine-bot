# After 30 Med Bot — Telegram бот для напоминаний о приёме лекарств

> ⚡ Вайб-кодинг проект — написан в паре с Claude (AI) за один вечер.
> Код живой, рабочий, итерируем дальше.

## Стек
- Python 3.14
- python-telegram-bot 22.7 — Telegram API (async)
- APScheduler 3.11.2 — планировщик напоминаний (каждую минуту)
- SQLite — база данных
- pytz + timezonefinder + geopy — определение часового пояса

## Структура проекта

```
med-bot/
├── bot.py              # точка входа — регистрация handlers, main()
├── database.py         # вся работа с SQLite (CRUD + миграции)
├── scheduler.py        # отправка напоминаний каждую минуту
├── constants.py        # состояния диалогов, метки, слоты времени
├── utils.py            # общие утилиты (декоратор ошибок, cancel)
├── handlers/
│   ├── meds.py         # добавление, редактирование, удаление лекарств
│   ├── stats.py        # статистика за сегодня и 7 дней
│   ├── settings.py     # настройки, пресеты времени, режим напоминаний
│   └── timezone.py     # /start, определение часового пояса, главное меню
├── .env                # токен бота (не коммитить!)
├── requirements.txt
└── med_bot.db          # SQLite база (создаётся автоматически)
```

## База данных

| Таблица | Описание |
|---|---|
| `users` | пользователи: telegram_id, timezone, reminder_mode, пресеты времени |
| `medications` | лекарства: название, дозировка, способ приёма, кол-во приёмов |
| `schedule_rules` | правила расписания: время, тип (daily / interval / weekdays / monthly) |
| `schedules` | устаревшая таблица, оставлена для совместимости |
| `intake_log` | история приёмов (taken / skipped / pending) |

### Типы расписания

| Тип | Описание |
|---|---|
| `daily` | каждый день |
| `interval` | каждые N дней от даты начала |
| `weekdays` | по выбранным дням недели (Пн–Вс) |
| `monthly` | раз в месяц, N-го числа |

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | регистрация, запрос часового пояса, главное меню |
| `/meds` | список лекарств + добавить / изменить / удалить |
| `/stats` | статистика за сегодня или за 7 дней |
| `/settings` | настройки: часовой пояс, режим напоминаний, время приёмов |
| `/about` | о проекте |
| `/cancel` | отмена текущего диалога |

## Функционал

- **Добавление лекарств** с дозировкой, способом приёма, гибким расписанием
- **Multi-select время приёма** — выбирай 🌅 Утро / ☀️ Обед / 🌇 Вечер / 🌙 Ночь; можно любую комбинацию
- **Персональные пресеты** — каждый пользователь настраивает своё время Утра/Обеда/Вечера/Ночи в /settings
- **Гибкое расписание** — каждый день, через N дней, по дням недели, раз в месяц
- **Напоминания** по расписанию с кнопками ✅ Принял / ❌ Пропустить
- **Два режима напоминаний**: один раз или повтор каждые 5 минут до подтверждения (до 2 часов)
- **Статистика за сегодня** с итогом и цветовым индикатором 🟢🟡🔴
- **История за 7 дней** с детализацией по дням и времени приёма
- **Определение часового пояса** по геолокации или городу
- **Поддержка нескольких пользователей**, лимит 10 лекарств на пользователя
