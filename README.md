# After 38 Med Bot — Telegram бот для напоминаний о приёме лекарств

> ⚡ Вайб-кодинг проект — написан в паре с Claude (AI) за один вечер.
> Код живой, рабочий, итерируем дальше.

## Стек
- Python 3.14
- python-telegram-bot — Telegram API
- APScheduler — планировщик напоминаний
- SQLite — база данных
- pytz + timezonefinder + geopy — определение часового пояса

## Структура проекта

```
med-bot/
├── bot.py              # точка входа — регистрация handlers, main()
├── database.py         # вся работа с SQLite (CRUD + миграции)
├── scheduler.py        # отправка напоминаний каждую минуту
├── constants.py        # состояния диалогов, метки
├── utils.py            # общие утилиты (декоратор ошибок, cancel)
├── handlers/
│   ├── meds.py         # добавление, редактирование, удаление лекарств
│   ├── stats.py        # статистика за сегодня и 7 дней
│   ├── settings.py     # настройки, режим напоминаний, /about
│   └── timezone.py     # /start, определение часового пояса
├── .env                # токен бота (не коммитить!)
├── requirements.txt
└── med_bot.db          # SQLite база (создаётся автоматически)
```

## База данных

| Таблица | Описание |
|---|---|
| users | пользователи (telegram_id, username, timezone, reminder_mode) |
| medications | лекарства (название, дозировка, способ приёма, кол-во раз) |
| schedules | времена напоминаний для каждого лекарства (HH:MM) |
| intake_log | история приёмов (taken / skipped / pending) |

## Команды бота

| Команда | Описание |
|---|---|
| /start | регистрация, запрос часового пояса |
| /meds | список лекарств + добавить / изменить / удалить |
| /stats | статистика за сегодня или за 7 дней |
| /settings | настройки: часовой пояс и режим напоминаний |
| /about | о проекте |
| /cancel | отмена текущего диалога |

## Функционал

- Добавление лекарств с указанием дозировки, способа приёма (натощак / после еды / во время / независимо), количества приёмов в день и времени
- Напоминания по расписанию с кнопками ✅ Принял / ❌ Пропустить
- Два режима напоминаний: один раз или повтор каждые 5 минут до подтверждения
- Статистика за сегодня с итогом дня и цветовым индикатором 🟢🟡🔴
- История за 7 дней с детализацией по дням и времени приёма
- Определение часового пояса по геолокации или городу
- Поддержка нескольких пользователей
